Intro to Programming Lesson 01 Notes

>>>How it Works
The web is a network of computers that communicate with one another via a
HTTP request to a server.
Once the request has been made the server sends the HTML document to
the computer that made the request, this is when the computer interprets the
HTML document via a web browser and displays it on the users screen.

>>> HTML
HTML stands for Hypertext Markup Language, which form the majority of
content on the Internet. HTML consists of text content to determine what you
see and markup content to determine how it looks.
+++ CSS is used to modify the visual aspect of the text to a greater extent +++

>>> HTML Vocabulary
Tag = An HTML tag is always contained within angled brackets
tags are used to tell browsers the type of each element.
Element = An element is everything within a set of opening and closing tags,
HTML documents are made up of these elements.
Attribute = This is a property of a HTML element.

>>>Types of Elements
HTML elements are either inline or block. Block elements form an invisibile box
arount the content, whereas inline elements do not.

>>>Important to Remember
In programming it is important not to forget that computers always translate the code
literally, thus there may be simple syntax errors that cause frustration. It is important
check for accuracy and keep calm when working on code!
